# palm2-tutotrial

Getting Started
0. Create a new project directory
Create a new project directory and navigate to it in your terminal:
```
mkdir palm2-tutorial
cd palm2-tutorial
```

Create and activate virtual environment:

```
bash inir_setup.sh
```

setup api

2. Run the app
```
streamlit run app.py
```
